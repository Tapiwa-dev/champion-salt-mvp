# Champion Salt Management Portal

MVP web application for Champion Salt operations. The portal gives staff a single interface for tracking inventory, orders, customers, warehouses, deliveries, invoices, payments, analytics, and system settings.

## MVP Scope

- Dashboard overview with key operational metrics and quick navigation.
- Inventory, orders, customers, warehouses, deliveries, invoices, and payments workspaces.
- Analytics view for sales and business reporting.
- Responsive React interface built with Vite, Tailwind CSS, Radix UI, and Lucide icons.

## Tech Stack

- React 18
- Vite 6
- Tailwind CSS 4
- Radix UI components
- Recharts
- Lucide React icons

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Client Notes

This MVP is frontend-ready for demonstration and feedback. The current data is UI/demo data; production deployment should connect the workflows to the selected backend, authentication, and live business data sources.
